This table contains automated diff's of files between moveset files for Ultra Street Fighter 4. It uses the fact that all files from previous patches are present on PC versions of the game to create a list of raw differences between the files. As a result, the resulting data can be a little complicated to understand, but its all accurate, and sometimes changes that Capcom doesn't list have ended up happening! Think of this page as a way to get insight into how these patches are coded and delievered to us, and a way to check for hidden tech that has popped up in the game that no one may know about! -dantarion

Shotouts to @Eternal0Star, who always hounds me about getting the patch diffs up quickly, as he is the one that takes this automated data and presents it to the world in laymans terms!
